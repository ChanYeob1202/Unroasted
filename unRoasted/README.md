## What It Does

Coffee Community is a space where I share my personal coffee journey and coffee enthusiasts can post their thoughts about coffee. The platform features community driven content and curated coffee insights for articles. 

## Why I Built This

As someone passionate about coffee, I wanted to create a space where people could share their coffee experiences and learn from each other. This project combines my love for coffee with my development skills.

## Features

- **Personal Coffee Journey**: Share my story and experiences with coffee
- **Community Posts**: Users can share their coffee thoughts and discoveries
- **Coffee Insights & Articles**: Curated articles about coffee culture, brewing methods, and more

## Tech Stack

- React.js
- Tailwind CSS
- React Router
- Firebase (Authentication)
- Strapi (CMS)

## Getting Started

```bash
cd micheal_codes_coffee

npm start
```
